# Coding Workflow

## Goal

Use this workflow when implementing or modifying code.

## Steps

1. Read README.md and AGENTS.md.
2. Inspect the current repository structure.
3. Check git status before making changes.
4. Keep changes scoped to the requested task.
5. Prefer existing project conventions.
6. Run available tests or checks.
7. Summarize changes, verification, and remaining risks.
