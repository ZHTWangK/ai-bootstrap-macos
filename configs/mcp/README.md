# MCP

This directory records Model Context Protocol configuration notes.

MCP is used to connect AI tools with external systems, tools, and data sources.

## Planned MCP Servers

- filesystem
- GitHub
- Docker
- browser/search
- cloud provider
- database
- documentation/search

## Principles

- Do not commit secrets.
- Prefer official or widely used MCP servers.
- Keep MCP configuration project-specific where possible.
- Add one MCP server at a time and verify it with doctor checks.
